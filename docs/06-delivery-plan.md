# Delivery Plan

## Phase 0: Discovery and Alignment

Outcomes:

- Confirm Salesforce org model, Experience Cloud license model, and source-of-truth systems.
- Confirm UChicago brand assets, approved logo files, and exact digital color values.
- Identify content manager groups and publishing workflow.
- Inventory existing Salesforce objects for alumni, events, giving, groups, preferences, and content.

Acceptance criteria:

- Object inventory exists.
- Integration inventory exists.
- Brand asset access confirmed.
- Launch audience and authentication model confirmed.

## Phase 1: Foundation

Outcomes:

- Experience Cloud site created in sandbox.
- Theme, header, footer, navigation, and brand tokens established.
- Content model configured for first content types.
- Permission sets and publishing workflow configured.
- Initial component contracts written.

Acceptance criteria:

- Content managers can draft, review, publish, schedule, and archive launch content.
- Portal only renders published, in-window content.
- Header/footer treatment passes brand review.
- Basic accessibility checks pass.
- Foundation work is validated in the POC sandbox site: https://uchicagoard--poc.sandbox.my.site.com/AlumniAndFriends

## Phase 2: Launch Experience

Outcomes:

- Personalized home page.
- News/story listing and detail.
- Event promotion and registration handoff.
- Giving promotion modules.
- Volunteer opportunity listing.
- Group directory.
- Profile and preferences view.

Acceptance criteria:

- Launch user journeys work for alumni and friends.
- Content managers can update launch pages without code changes.
- Portal profiles cannot view internal-only fields.
- Mobile and desktop layouts pass review.

## Phase 3: Reporting and Hardening

Outcomes:

- Content and engagement dashboards.
- Broken link and expired content process.
- Performance, accessibility, and security hardening.
- UAT feedback resolved.

Acceptance criteria:

- UAT signoff complete.
- Accessibility review complete.
- Security review complete.
- Production deployment plan approved.

## Phase 4: Enhancements

Candidate enhancements:

- Mentorship matching.
- Career resources.
- Deeper personalization.
- Email marketing integration.
- Advanced analytics.
- Group-specific delegated content management.
