# Salesforce Source

This folder contains Salesforce metadata for the alumni and friends Experience Cloud solution.

## Boundaries

| Area | Folder | Notes |
| --- | --- | --- |
| Lightning Web Components | `main/default/lwc` | Reusable portal presentation and record-driven content components. |
| Custom objects and fields | `main/default/objects` | Portal-owned content model metadata. |
| Permission sets | `main/default/permissionsets` | Content manager and portal operations access. |
| Custom labels | `main/default/labels` | User-facing labels that may need governance or localization. |

## Contributor Rule

Before changing metadata, read the closest local `README.md` and the relevant planning doc under `docs/`.
